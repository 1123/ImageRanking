Implicit Feedback Ranking by Acyclic Graph Maintenance

This page shows how the attractiveness of images can be learned from implicit feedback from users to a website.
  1. Enter a search term below to search for tagged images on FlickR.
  2. A random sample of the retrieved images is displayed.
  3. Select an image that you like by clicking on it. A json call will be issued to the server, including the list of ids of displayed images plus the selected image. This is the implicit feedback shopping sites or social networks generate by users navigating the site.
  4. Navigate back to this page, which will display a different set of pictures for the search term.
  5. Again, select an image that you like.
  6. Repeat the above steps a couple of times.

Related Work:

  1. BPR: Bayesian Personalized Ranking from Implicit Feedback
    This work describes an approach to personalized ranking. This creates a user specific ranking for a set of items. We do not do personalized ranking, since we are interested in the objective attractiveness of images. Knowing the overall attractiveness is important for
    • Showing the page to possible investors.
    • Improving the first impression of your site for first time customers.
    • Giving your content providers information on how their images compare to others.
